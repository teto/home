* ISSUES

** `WARNING:  database "postgres" has a collation version mismatch`

collation refers to locales/glibc etc
```
template1=# show lc_collate;
 lc_collate  
-------------
 en_US.UTF-8
(1 row)
```

Dont forget to add the ";"
```
ALTER DATABASE template1 REFRESH COLLATION VERSION;
```
